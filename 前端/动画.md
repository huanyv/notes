# 动画

## 打字机动画

* <https://www.cnblogs.com/Aaxuan/p/14146418.html>

```
<span id="subtitle"></span>
<script src="https://cdn.jsdelivr.net/npm/typed.js@2.0.11"></script>
<script>
    var typed = new Typed("#subtitle", {
        strings: ['文本1', '文本2'],
        startDelay: 300,
        typeSpeed: 100,
        loop: true,
        backSpeed: 50,
        showCursor: true
    });
</script>
```
